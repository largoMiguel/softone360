# SOLUCTIONS

Repositorio con backend (Python/Flask) y frontend (Angular).

Estructura principal:
- backend/: código del servidor
- frontend/: aplicación Angular

Pasos para subir a GitHub (si ya tienes un repo remoto):
1. git remote add origin <URL-del-repositorio>
2. git branch -M main
3. git push -u origin main

Si quieres que yo cree el repositorio remoto en GitHub, instala la GitHub CLI (`gh`) y autentícate, o proporciona la URL/remoto. También puedo asesorarte para crear el repo desde la web.
